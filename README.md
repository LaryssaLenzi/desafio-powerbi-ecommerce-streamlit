# 📊 Dashboard E-commerce - Desafio DIO

Este projeto foi desenvolvido como parte de um desafio da **DIO (Digital Innovation One)**. O objetivo original era realizar modelagem e transformação de dados com DAX no Power BI, mas decidi elevar o desafio construindo uma **aplicação web interativa completa utilizando Python e Streamlit**.

## 🔗 Link para o Projeto Online
🚀 **[Clique aqui para acessar o Dashboard em Tempo Real](https://desafio-powerbi-ecommerce-app.streamlit.app/)**

---

## 🛠️ Tecnologias Utilizadas
* **Python**: Linguagem base do projeto.
* **Streamlit**: Framework para criação do dashboard web.
* **Pandas**: Para limpeza, transformação e modelagem dos dados (substituindo o Power Query/DAX).
* **Plotly**: Para a criação de gráficos dinâmicos e interativos.

## 📈 Funcionalidades
- **KPIs Dinâmicos**: Visualização instantânea de Receita Total, Ticket Médio e Volume de Pedidos.
- **Filtros Interativos**: Segmentação por Categoria de Produto e Status do Pedido via barra lateral.
- **Análise Temporal**: Gráfico de evolução mensal para identificar tendências de vendas.
- **Visão Logística**: Monitoramento do status de entrega e distribuição de tamanhos (inventário).
